Crea il progetto Firebase
Gratuito — account Google richiesto
1. Clicca il bottone qui sotto per aprire Firebase Console
2. Clicca "Aggiungi progetto"
3. Nome progetto: caffe-fedelta
4. Disattiva Google Analytics (non serve)
5. Clicca "Crea progetto" → attendi → "Continua"
🔥 Apri Firebase Console

Hai già un progetto Firebase?

🗄️
Crea il Realtime Database
Il database dove vengono salvati i punti
1. Nel tuo progetto Firebase, clicca il bottone qui sotto
2. Nel menu a sinistra clicca "Realtime Database"
3. Clicca "Crea database"
4. Scegli regione "europe-west1 (Belgium)"
5. Seleziona "Inizia in modalità test"
6. Clicca "Abilita"
🗄️ Apri Firebase Console
🔒
Imposta le regole di accesso
Permette all'app di leggere e scrivere
1. Nel Realtime Database clicca il tab "Regole"
2. Sostituisci tutto il testo con quello qui sotto:
{
  "rules": {
    "bars": {
      "$uid": {
        ".read": true,
        ".write": "auth != null && auth.uid == $uid"
      }
    }
  }
}
3. Clicca il bottone "Pubblica" in alto a destra
🔒 Apri Firebase Console
🔑
Copia la API Key
L'ultima cosa — promesso!
1. Clicca il bottone qui sotto
2. Clicca l'icona ⚙️ "Impostazioni progetto" (in alto a sinistra)
3. Scorri in basso fino a "Le tue app"
4. Se non hai app web: clicca "</>" → nome caffefedelta → Registra
5. Trova il campo apiKey nel codice mostrato
6. Copia il valore (inizia con AIzaSy...)
7. Incollalo qui sotto 👇
